# Repository agent notes

## Gemini static pages

When the task is about `Gemini/`, read `Gemini/AGENTS.md` before editing.

The generated root HTML files in `Gemini/` are browser entry points, but the maintainable source of truth is under `Gemini/src/`, `Gemini/styles/`, `Gemini/scripts/`, `Gemini/assets/`, and `Gemini/tools/`.

Do not rebuild the pages as one large hand-edited HTML file.
